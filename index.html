Non-deterministic Finite Automaton (NFA)

Instructions

Create the NFA you wish to convert to a DFA. Or, you can import a saved NFA, or even use one from a list of presets. Start building the NFA by right-clicking on the canvas below to create a node. Then, right-click on the new node to create a transition, edit its properties, or delete it. You can also pan and zoom the canvas.

Deterministic Finite Automaton (DFA)

Instructions

After building the NFA, you can then start converting it to a DFA. This can be done in steps , animated , or all at once . You may drag nodes around to reorganize the visual layout. However, node-editing features are disabled. To change the DFA, you must first change the NFA.
